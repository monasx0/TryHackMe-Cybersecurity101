# Windows Fundamentals

A beginner-friendly guide to Windows OS tools, system management, and security features.

---

## Overview
This documentation covers three main areas:
1. **Part 01** – Introduction & Core Utilities  
2. **Part 02** – System Configuration & Troubleshooting Tools  
3. **Part 03** – Windows Security & Protection Features  

---

## Key Tools & Features

- **MSConfig** – Startup and boot troubleshooting  
- **Task Scheduler / Event Viewer** – Automate tasks and monitor system events  
- **Resource Monitor** – Track CPU, memory, disk, and network usage  
- **Command Prompt (`cmd`)** – Basic system and network commands  
- **Registry (`regedit`)** – System and application settings  
- **Windows Security** – Virus protection, Firewall, SmartScreen  
- **Core Isolation & TPM** – Hardware security and memory protection  
- **BitLocker** – Drive encryption  
- **Volume Shadow Copy Service (VSS)** – Backup and restore system files  

---

## What You Will Learn

- Manage **Windows Update** and security patches  
- Navigate **Windows Security** protection areas  
- Configure **Virus & Threat Protection**  
- Control **Firewall & Network Protection**  
- Understand **SmartScreen**, Core Isolation, and **TPM** 
- Use **BitLocker** and **VSS** for data protection
